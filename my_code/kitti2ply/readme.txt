KITTIデータの点群データ（.bin）をplyに変換するやつ

使い方：
bash kitti2ply.sh [binが入ってるディレクトリまでのパス] [保存先のディレクトリ名] [変換するデータの最初の番号]　[変換するデータの最後の番号]

保存先のディレクトリ名：plyファイルは，実行ディレクトリの"ply"の中に作られる．plyの中でディレクトリを分けるためにここで適当に指定
変換するデータの最初の番号：部分的に変換したいならbinファイルの番号を適当に指定　最後の番号も同様
    全部にかけたいなら　最初の番号に　0　を指定して最後をでかい数字にすれば，全部かけ終わったタイミングでエラーで止まると思う
